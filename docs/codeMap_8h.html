Remove text interval.

Parameters
inS-> input text.
pos-> initial position.
limit-> limiting string.
eraseLimit-> Enable/disable limit erasing.
scapped-> Enable/disable considere scapped characters for limits.
Returns
Return 0 if the limiting string has been found.
Return 1 if the limiting string has not been found.
Return -1 if the limiting string has non valid length.

Search in inS string from position pos+1 until end of string the specified limiting string limit. All characters in this interval will be removed. If eraseLimit is true, the limiting string will be removed too. If scapped is true, limit will be considered as valid limit. Otherwise, if is false, limit will not be a valid limit match.

The first position to be erased is pos.
